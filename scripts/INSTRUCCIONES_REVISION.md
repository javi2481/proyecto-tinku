# Instrucciones para Revisión Pedagógica — Phase 3

## Dónde hacer la revisión

1. **Ir a**: https://tinku.app/review-exercises
2. **Login** con tu cuenta de padre/admin

## Cómo revisar cada ejercicio

### Paso 1: Leer el ejercicio
- Mirá el prompt (la pregunta)
- Mirá las opciones
- Mirá la respuesta correcta

### Paso 2: Aplicar la guía pedagógica
Consultá `memory/pedagogical-guide.md` para los criterios:

| Score | Definición |
|-------|------------|
| **5** | Ejercicio perfecto, listo para producción |
| **4** | Muy bueno, mínimos detalles |
| **3** | Aceptable, funciona |
| **2** | Problemas significativos |
| **1** | No usable, error fundamental |

### Paso 3: Asignar estrellas
- Hacé click en las estrellas ⭐ (1-5) para asignar
- O usar los botones de Approve/Reject

## Atajos

### Bulk approve rápido
Si un concepto tiene 8+ ejercicios que看起来 bien:
- Asigná score 4 o 5 a cada uno
- Click en "Approve" para marcar como approved

### Checklist rápido
Para cada ejercicio, preguntate:
1. ¿La pregunta es clara para un nene de 6-12 años?
2. ¿La respuesta correcta es objetivamente correcta?
3. ¿Los distractores son todos plausibles (ninguno obviously wrong)?
4. ¿El nivel de dificultad coincide con el grade?

## Tiempo estimado
- ~15-20 segundos por ejercicio
- ~331 ejercicios → ~1.5 - 2 horas
- Recommiendo hacer en batches de 20-30 ejercicios

## Guardar progreso
No hay necesidad de guardar — los scores se guardan automáticamente en la DBcuando asignás las estrellas.
